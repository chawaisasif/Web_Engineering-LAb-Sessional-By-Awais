
# 🎓 Mini Student Web Portal

A simple multi-page **Student Portal Web Application** developed using **HTML, CSS, Bootstrap, and JavaScript**.
This project is created as part of the Web Engineering Lab sessional.

---

## 📌 Features

* ✅ Student Registration Form with Validation
* ✅ Dashboard with Welcome Message
* ✅ Show Current Date & Time
* ✅ Course Selection System
* ✅ Feedback Form with Validation
* ✅ Fully Connected Multi-Page Navigation
* ✅ Responsive UI using Bootstrap

---

## 🗂️ Project Structure

```
📁 Student-Portal
│── index.html        # Home Page
│── registration.html # Registration Page
│── dashboard.html    # Dashboard Page
│── courses.html      # Courses Page
│── feedback.html     # Feedback Page
│── style.css         # Styling File
│── script.js         # JavaScript Logic
```
